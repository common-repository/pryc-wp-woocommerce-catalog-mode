== Description ==
For WooCommerce plugin - hide "add to cart" button and price (single product and product list/archive).

Just active plugin to start "catalog mode" (hide "add to cart" and price).

== Installation ==
1. Upload `pryc-wp-woocommerce-catalog-mode` dir to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
